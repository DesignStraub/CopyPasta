# CopyPasta
Clipboard History Tool with additional functions.
100% Offline, History is saved in %Appdata%/Roaming/Copypasta/CbList.json

### Usage:
Ctrl+Shift+V Open.
Click on any Entry to enter it to the next Textinput you click.
